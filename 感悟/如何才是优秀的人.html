# 如何才是优秀的人

# 什么样的人才是优秀的人？

初中时代，我看了很多人物传记，像罗斯福，丘吉尔等。。。感觉他们为国家作出贡献，是很优秀的人。我也励志要成为一个像他们一样的人，后来发现时势造英雄，不是所有人能像他们一样伟大的。 ** 大学期间，接触了编程，看过乔布斯转，微软的传记，浪潮之巅，人生神话，地狱黑客，等。这些书籍对我触动很大，我觉得他们也属于那类改变了世界的人。也因此让我对编程充满了兴趣。想通过努力，成为一位像他们一样的人。经历过大学的几年拼搏，我发现他们属于天才型选手，好像也是不是通过努力就可以到达的人生高度。 ** 工作期间，看到当今社会中的大牛，像阮一峰，尤大神，司徒正美等。。。我觉得至少我得像他们一样优秀吧！！！于是，我每天上下班之余，都会给自己充充电，学习起来。希望自己能够可以达到他们的高度。 ** 在我拥有这些想法的时候，我天天都会很努力的实现自己的目标。为此，我抛弃了很多东西。比如说:与朋友的相处，与父母的沟通。而且我相信一句话：人应为优秀而孤独。因此，跟我相处的人，都会觉得我高冷，我也不会主动联系他们（包括父母）。可能父母也会觉得养了一个白眼狼吧.....

直到昨天，2019.7.7 凌晨，我和我媳妇聊天，他想让我多和父母联系沟通起来，我说和父母没啥沟通语言。然后我媳妇跟我说了一句话：子欲孝而亲不在。对我触动挺大的。其实，父母对我们并无所求，只是想知道我们活着好不好，开不开心什么的，难道这点时间都腾不出来么？

这件事情使我联想到了另一个事情，有一天，我和老板一块吃饭，谈起了二战，我一直认为二战促进了科技的发展与进步。老板回答：是的，但是促进科技的发展，不是战争，而是竞争，只要有良好的竞争，科技发展照样可以很快。这个观点对我触动其实也很大的。让我提出疑问：促进科技的发展有很多方法，为什么非要选择战争呢？

感觉老板也是一个很优秀的人，一下子可以认识到事物的本质，而我这么多年，却还只是看到事物的表面现象。

在这一晚上中，让我想明白很多，其实我就是一个普通的人。没有必要要求自己成为一个自己理想中的人，那样本身就是不切实际的。在这个过长夜中，我也渐渐想明白了，人不是因为优秀而优秀，是因为信仰而优秀。

# 什么是优秀？

  1. 在妻子眼里是一个好丈夫。
  2. 在孩子眼是是一个好爸爸。
  3. 在父母眼里是一个好儿女。
  4. 在公司眼里是一个好员工。
  5. 在国家眼里是一个好公民。

优秀其实就是你把你当面所在的位置尽自己最大的可能做得最好，那便是优秀。

# 总结：

  1. 当儿女需要你保护的时候，你站出来，那便是优秀。
  2. 当公司面临困境的时候，你站出来了，那便是优秀。
  3. 当国家面临侵略的时候，你站出来了，那便是优秀。

# 人生计划

  1. 去旅游的每个地方，下火车的时候，要拍照几年，60岁统计下祖国的大好河山走过那些。
  2. 多和父母，朋友沟通交流。
  3. 喜欢自己的工作，并且努力专研他。
  4. 控制住自己的脾气，不要被情绪所控制。
  5. 善待他们，不要说刻薄的话。
